等宽（等分布局）的float方法

margin-left: -20px;是为了给父容器增加间距（需要和间距一致），从而方便切分
box-sizing: border-box;是为了让宽度包含padding
缺点：结构和样式有耦合性（改变等分数量要改变css样式）


<div class="parent">
    <div class="column"><p>1</p></div>
    <div class="column"><p>2</p></div>
    <div class="column"><p>3</p></div>
    <div class="column"><p>4</p></div>
</div>

    .parent{
    margin-left: -20px;
}
.column{
    float:left;
    width: 25%;
    padding-left: 20px;
    box-sizing: border-box;
    }
p{
    background:#fe8019;
}
