Demo

absolute+transform水平垂直居中

适用于移动端
不影响其他元素


<div class="parent">
    <div class="children">Demo</div>
</div>


.parent {
    position:relative;
    width:100%;
    height:300px;
    background:#eee;
}
.children {
    position:absolute;
    left:50%;
    top:50%;
    width:50px;
    height:50px;
    background:#e60415;
    transform:translate(-50%,-50%);
}
