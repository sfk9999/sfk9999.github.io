淘宝双飞翼布局

思路：
1. 三者都设置向左浮动
2. 设置main宽度为100%
3. 设置负边距：left设置负边距为负的100%，right设置负边距为负的自身宽度
4. 设置main-Inner的margin值给左右两个子面板留出空间。


<div class="container">
    <div class="main">
        <div class="mainInner">main</div>
    </div>
    <div class="left">left</div>
    <div class="right">right</div>
</div>


.main,.left,.right{
    float:left;
    min-height: 100px;
}
.main{
    width: 100%;
    background: #fe8019;
}
.left{
    margin-left: -100%;
    width: 200px;
    background: #fef20d;
}
.right{
    margin-left: -220px;
    width: 220px;
    background: #ffcc66;
}
.mainInner{
    margin-left: 200px;
    margin-right: 220px;
}
