的垂直水平居中
absolute+transform
多行文字
的垂直水平居中flex方法 

在父元素浮动的情况下进行单行和多行文字的居中

因为有float，所以不能使用display:table-cell;的方法。


<div class="parent">
    <span class="single">一行文字</span>
</div>
<div class="parent parent_multiVertical">
    <span class="multiVertical">多行文字
    <br>的垂直水平居中
    <br>absolute+transform</span>
</div>
<div class="parent parent_multiVertical_flex">
    <span>多行文字<br>的垂直水平居中flex方法</span>
</div>


.parent{
    float: left;
    margin-right: 20px;
    width: 200px;
    height: 200px;
    border:1px solid red;
    text-align: center;
}
.single{
    line-height: 200px;
}
.parent_multiVertical{
    position: relative;
}
.multiVertical{
    position: absolute;
    top:50%;
    left: 50%;
    transform: translate(-50%,-50%);
}
.parent_multiVertical_flex{
    display: flex;
    justify-content: center;
    align-items: center;
}
