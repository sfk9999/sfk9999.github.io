等宽（等分布局）的table方法

没设置单元格宽度的原因：table-layout: fixed;(布局优先；若单元格未设置宽度，则宽度均分)
缺点：多列一些结构代码


<div class="parent-fix">
    <div class="parent">
        <div class="column"><p>1</p></div>
        <div class="column"><p>2</p></div>
        <div class="column"><p>3</p></div>
        <div class="column"><p>4</p></div>
    </div>
</div>


.parent-fix{
    margin-left: -20px;
}
.parent{
    display: table;
    width: 100%;
    table-layout: fixed;
}
.column{
    display: table-cell;
    padding-left: 20px;
}
p{
    background:#fe8019;
}
